Gerenciamento de Pastas com Python

Análise de Código e Manipulação do Sistema Operacional

Visão Geral do Projeto

Este projeto consiste num script em Python desenvolvido para automatizar o controlo de acessos e a organização de diretórios no sistema operacional. Ele utiliza um arquivo de texto local como um banco de dados simples para validar utilizadores permitidos e, em seguida, oferece opções para criar ou remover pastas automaticamente.

Análise Detalhada do Código

1. Interface Inicial e Limpeza de Terminal

O programa inicia importando a biblioteca nativa os, essencial para interagir diretamente com o sistema operacional. Logo após receber o nome do utilizador, o comando os.system('cls' if os.name == 'nt' else 'clear') é executado para limpar o ecrã, garantindo uma interface CLI (Interface de Linha de Comando) limpa e profissional.

2. O Banco de Dados de Utilizadores

O arquivo Usuarios.txt guarda os nomes das pessoas autorizadas. No código, a segurança e leitura correta são feitas através de:

  • Verificação de Existência: O comando os.path.exists('Usuarios.txt') impede que o programa vá abaixo caso o arquivo mestre desapareça.
  • Gerenciamento de Contexto: O bloco with open(...) abre o arquivo de forma segura e garante o seu fechamento automático.
  • Tratamento de Codificação: O parâmetro encoding='utf-8' é configurado para suportar caracteres acentuados sem quebrar o código.
  • List Comprehension e Sanetização: A expressão [linha.strip() for linha in file.readlines()] lê as linhas e remove os espaços ou quebras de linha invisíveis (\n), evitando falhas na comparação dos nomes.

3. Sistema de Autenticação e Bloqueio

Após ler a lista, o script valida se o input inserido está presente no arquivo. Se o utilizador não for encontrado, a função exit() encerra o programa imediatamente, funcionando como um verdadeiro porteiro do sistema.

4. Automação de Diretórios (Criar e Apagar)

Com o utilizador devidamente validado, o script executa tarefas operacionais automatizadas:

  • Criação Segura: Utiliza os.mkdir() para gerar uma pasta personalizada com o nome do utilizador logado, mas apenas após confirmar com o os.path.exists() que ela ainda não existe.
  • Listagem de Arquivos: O comando os.listdir() exibe em tempo real no terminal tudo o que está na pasta do projeto para que o utilizador possa escolher o que apagar.
  • Eliminação de Pastas: O comando os.rmdir() é acionado caso o utilizador decida excluir alguma pasta listada.

🚀 Aprendizados Adquiridos

O desenvolvimento deste script em Python consolidou conceitos cruciais de lógica e programação backend:

  1. Manipulação de Ficheiros (I/O): Leitura detalhada de arquivos de texto e limpeza de strings.
  2. Integração com o Sistema Operacional: Domínio sobre comandos de terminal, navegação e manipulação de pastas via código.
  3. Tratamento de Exceções e Fluxos: Prevenção de erros operacionais comuns (como tentar abrir arquivos inexistentes) através de verificações prévias.
