Automação de Formas Geométricas

RPA, Trigonometria Aplicada e Simulação de Interface (Paint)

Visão Geral do Projeto

Este projeto consiste num script de automação de interface (RPA) desenvolvido em Python que abre o Microsoft Paint automaticamente e utiliza as coordenadas do rato para desenhar formas geométricas (Quadrado, Triângulo e Círculo) sozinho, baseando-se na escolha do utilizador.

Análise Detalhada do Código

1. As Bibliotecas Utilizadas

O script depende de um conjunto de bibliotecas integradas e externas para executar o controlo periférico:

  • pyautogui: Responsável por assumir o controlo do rato e do teclado, permitindo simular cliques, arrastos e digitação como se fosse uma pessoa real.
  • time: Utilizada para inserir pausas estratégicas (time.sleep), garantindo que o sistema operativo consiga carregar as janelas antes do Python enviar comandos.
  • math: Fornece as funções de cálculo matemático necessárias para converter graus em radianos e mapear curvas na tela.

2. As Funções de Desenho

Cada forma geométrica utiliza uma abordagem lógica diferente para movimentar o cursor pela tela:

  • O Quadrado: Utiliza o comando dragRel para fazer arrastos relativos à posição atual do rato, movendo 100 píxeis sequencialmente nas direções direita, cima, esquerda e baixo.
  • O Triângulo: Calcula a base reta e, em seguida, recua metade do tamanho horizontal enquanto sobe no eixo vertical para criar o vértice central, fechando a diagonal de forma precisa.
  • O Círculo Perfeito: Utiliza trigonometria avançada (cosseno para o eixo X e seno para o eixo Y). Através de um loop que percorre 360 graus de 15 em 15, a função usa dragTo com uma velocidade quase instantânea (duration=0.01) para conectar os pontos da circunferência de forma suave.

3. Inicialização e Loop de Decisão

O fluxo principal do programa automatiza tarefas comuns e interage com o utilizador:

  • Abertura Automática: O script pressiona a tecla Windows, digita 'Paint' caractere por caractere e pressiona Enter, aguardando 3 segundos até que a ferramenta de desenho esteja pronta.
  • Caixa de Diálogo Gráfica: Utiliza a função pyautogui.confirm para exibir um menu pop-up customizado no ecrã.
  • Estrutura Match/Case: O programa avalia a string retornada pelo botão clicado e direciona o fluxo para a função geométrica correspondente de forma limpa.

🚀 Aprendizados Adquiridos

O desenvolvimento desta automação trouxe grandes aprendizados em lógica prática:

  1. Robótica de Software (RPA): Entendimento de como mapear interações humanas básicas de teclado e rato dentro de softwares do sistema.
  2. Matemática e Geometria no Código: Aplicação real de conceitos trigonométricos para a resolução de problemas de renderização e desenho no ecrã.
  3. Controlo de Sincronismo: Ajuste de temporizações e durações de movimento para evitar conflitos de velocidade entre o código e o hardware do PC.
