# 🚀 Meu Primeiro Portfólio Web

Seja bem-vindo(a) ao repositório do meu primeiríssimo portfólio pessoal! Este projeto foi desenvolvido com muito orgulho para marcar o início da minha jornada oficial como desenvolvedor, servindo como uma vitrine viva para reunir, organizar e demonstrar as minhas habilidades técnicas, os meus aprendizados e os principais projetos práticos que criei.

A ideia principal foi construir uma interface limpa, moderna e totalmente responsiva, onde futuros clientes, parceiros e recrutadores possam conhecer quem eu sou, o que eu domino e ver o meu código em ação.

---

## 🛠️ Tecnologias Utilizadas

Para a construção deste portfólio, foquei em consolidar fortemente as bases do desenvolvimento web front-end, garantindo uma estrutura sólida e uma estilização rica em detalhes:

* **HTML5:** Estruturação semântica de todas as páginas, seções e caminhos de navegação do site.
* **CSS3:** Estilização avançada com o uso de layouts modernos (Flexbox), efeitos dinâmicos de transição (`transition`), pseudo-classes (`:hover`), gradientes radiais espaciais no fundo e sombras projetadas (`box-shadow`) para dar profundidade ao design.
* **Mobile First & Responsividade:** Implementação de regras exclusivas via Media Queries (`@media`), adaptando fontes, espaçamentos internos (`padding`), quebras de texto em códigos (`word-break`) e o fluxo dos blocos para que o site fique impecável tanto em computadores quanto em celulares.

## 📈 Aprendizados Conquistados

Sendo o meu **primeiro portfólio**, o processo de desenvolvimento trouxe desafios reais de engenharia de software que expandiram a minha visão técnica:
1. **Arquitetura de Arquivos e Deploy:** Organização rígida de caminhos relativos de arquivos na raiz e subpastas para garantir a publicação correta no ambiente de produção do GitHub Pages.
2. **Resolução de Conflitos de Layout:** Domínio prático de propriedades do Flexbox (como `flex-direction` e alinhamentos) para corrigir problemas de elementos esmagados ou desalinhados em telas de tamanhos diferentes.
3. **Casamento entre Lógica e Design:** Aprender a documentar códigos complexos de Python de uma forma visualmente atraente e fácil de ler para o usuário final através da web.
